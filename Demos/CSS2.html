CSS Fonts & Images

Types of Fonts

  • Serif
  • Sans-serif
  • Monospace
  • Cursive
  • Fantasy

Serif fonts are best used for print media. Sans-serif fonts are best used for screen media. You can use Sans-serif fonts for a paragraph and serif fonts for headings.

Font-Size

  • xx-small, x-small, small, medium, large, x-large, xx-large
  • pixel
  • point
  • em
  • %

Font Properties

Font-Weight
Sets how thick or thin text displays (e.g., bold).
Font-Style
Specifies style (e.g., italics or oblique).
Line-height
Specifies line height (normal, %, or a number like 1.6).
Text-align
Sets text alignment: left, center, right, or justified.
Text-decoration
Options: none, underline, overline, or line-through.
Text-indent
Indents the first line of text in a block.
Text-transform
Controls capitalization: uppercase, lowercase, capitalize.
Letter-spacing
Adjusts spacing between characters.

Images

  • GIF - lossless, transparency, animation, 256 colors
  • JPG - lossy, no transparency, no animation, millions of colors
  • PNG - lossless, transparency, no animation, millions of colors

List Markers with CSS

list-style-type allows styling with none, disc, circle, square, decimal, upper-alpha, lower-alpha, lower-roman.

list-style-image allows you to use an image as the bullet.

list-style-position lets you state if the position is inside or outside.

Favorites Icon

A favorites icon (also known as favicon) adds an icon to the browser tab.
